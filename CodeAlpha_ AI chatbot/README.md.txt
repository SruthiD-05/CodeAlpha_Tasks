# AI Chatbot

## Overview
The AI Chatbot is a console-based Java application developed as part of the CodeAlpha Internship program. 
The system simulates human-like interaction using predefined conversational logic and basic input handling techniques.

## Features
- Interactive command-line conversation
- Pattern-based response system
- Chat history storage
- Modular code structure
- Error handling for invalid inputs

## Technology Stack
- Java (JDK 8 or above)
- Object-Oriented Programming (OOP)
- File Handling (for chat history persistence)

## Project Structure
AIChatbot.java
ChatbotEngine.java
chat_history.txt

## Installation & Execution

1. Clone the repository
2. Navigate to the project directory
3. Compile the program:

   javac AIChatbot.java

4. Execute the program:

   java AIChatbot
