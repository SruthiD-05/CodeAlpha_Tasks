# Stock Trading Platform

## Overview
The Stock Trading Platform is a Java-based simulation system designed to replicate fundamental stock market trading operations. 

The application allows users to buy and sell stocks, track portfolio performance, and manage transaction records through a structured console-based interface.

This project demonstrates the implementation of Object-Oriented Programming principles, financial transaction logic, and data management techniques in Java.

## Key Features
- Buy and sell stock functionality
- Portfolio tracking
- Transaction recording system
- Real-time balance updates
- Input validation and error handling
- Modular class-based architecture

## Technology Stack
- Java (JDK 8 or above)
- Object-Oriented Programming (OOP)
- Collections Framework
- Exception Handling

## Project Structure
StockTradingPlatform.java  
Stock.java  
Portfolio.java  
Transaction.java  

(Compiled `.class` files are generated during compilation and are excluded from version control.)

## Installation & Execution

1. Navigate to the project directory.
2. Compile the Java files:

   javac StockTradingPlatform.java

3. Run the application:

   java StockTradingPlatform

## System Workflow
1. User selects trading operation (Buy / Sell / View Portfolio).
2. System validates input and stock availability.
3. Portfolio balance is updated accordingly.
4. Transaction details are recorded and displayed.
